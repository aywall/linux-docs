[Normal Mode]

	:w saves a file
	:q exit a file
	:qa! - exit all files - changes lost

	i a - insert text before after cursor
	I A - inser text before after line
	ge e - end of word/token left, right
	{ } begining of previous, next paragraph
	( ) begining of prevoius, next sentence
	a - insert text after cursor
	A - insert text after last char on line

	v - highligt text
	d - cut text
	y - copy text
	p - past text

	c - change
	s - substitute
